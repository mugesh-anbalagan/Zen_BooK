## üìñ About Zen Book

Zen Book is designed for readers who want a distraction-free way to manage their reading list and track progress. We leverage the power of **Flutter** to deliver a fast, natively compiled, and consistently beautiful user experience across mobile platforms.

The goal is simplicity. Track your books, set reading goals, and see your reading history at a glance‚Äîall within a clean, minimalist interface.

## üöÄ Features

* **Simple Book Management:** Add books by title, author, and total pages. *Future: Integration with Open Library API for book lookup.*
* **Reading Progress Tracking:** Easily update your current page number to track progress.
* **Categorized Lists:** Organize books into three main lists:
    * **Currently Reading**
    * **Want to Read**
    * **Finished**
* **Reading Statistics:** View your reading habits (pages read per month, total books completed, etc.) in a clean dashboard.
* **Dark Mode Support:** A pleasant reading experience, day or night.
* **Local Persistence:** All your data is securely stored locally on your device using Hive/Sqflite.

## üíª Getting Started

This guide will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Flutter SDK installed on your machine.
* [Flutter SDK](https://flutter.dev/docs/get-started/install)
* [Dart SDK](https://dart.dev/get-dart)
* A suitable IDE (VS Code or Android Studio) with the Flutter and Dart plugins.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/mugesh-anbalagan/Zen_BooK.git](https://github.com/mugesh-anbalagan/Zen_BooK)
    cd zen_book
    ```
2.  **Get dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run the app:**
    ```bash
    flutter run
    ```
    * *Note: Ensure you have an active device or emulator connected.*
